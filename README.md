# 🐱☕ Cat Café Tower · 캣 카페 타워

귀여운 모찌 고양이가 팬케이크 탑을 오르는 하이퍼캐주얼 웹 게임입니다.
탑을 돌려 팬케이크의 **빈틈(khe hở)** 을 맞추고, 탭으로 점프해 위로 올라가세요.
콤보를 쌓으면 **🔥 파이어 모드** 가 발동해 모든 팬케이크(불탄 팬케이크 포함)를 뚫고 지나갑니다.

> Phaser 3 · 순수 HTML/JS (빌드 과정 없음) · 모바일 우선 · 모든 UI/콘텐츠 한국어

---

## ▶️ 실행 방법

**가장 간단한 방법 — 그냥 열기:**
`index.html` 을 크롬/사파리에서 더블클릭하면 바로 플레이됩니다.
(게임 코드는 ES 모듈이 아닌 일반 `<script>` 로 로드되어 `file://` 에서도 동작합니다. Phaser 는 CDN, 이미지는 로컬에서 로드됩니다.)

**로컬 서버로 실행 (권장):**
```bash
cd "Cat Cafe"
python -m http.server 8123
# 브라우저에서 http://localhost:8123/index.html
```

**배포:** `Cat Cafe` 폴더 전체(`index.html` + `src/` + `assets/`)를 GitHub Pages / Netlify 에 정적 업로드하면 끝입니다.

## 🎮 조작법

| 입력 | 동작 |
|---|---|
| **가로 드래그** | 탑 회전 (빈틈 정렬) |
| **탭 / 클릭** | 위로 점프 |
| **길게 누르기(≥300ms)** | 슬로우 모션 1초 (런당 3회, 콤보 5마다 1회 충전) |

### 규칙
- 위 팬케이크의 **빈틈을 아래(고양이 쪽)로 맞춘 뒤 탭** → 통과 성공, 콤보 +1
- **퍼펙트 점프**(착지 직후 빠른 탭) → 콤보 +2, 스파클
- 빈틈을 못 맞춰 **가장자리에 부딪히면** → 콤보 초기화 (화면 흔들림 + 연기)
- **불탄 팬케이크(빨강)** 의 가장자리에 부딪히면 → **게임 오버** (빈틈으로 깔끔히 통과하면 안전)
- **콤보 3 이상** → 🔥 파이어 모드 3초: 빈틈과 상관없이 모두 돌파
- **민트 팬케이크** → 점프 부스트, 자동으로 한 층 더
- **10층마다** 배경 테마 변경 (아침 → 노을 → 밤하늘 → 새벽)

### 점수
```
점수 = 높이 × 10 + 최대 콤보 × 50 + 퍼펙트 × 20
```
최고 점수는 `localStorage` 에 저장됩니다.

### 난이도 (엔드리스)
| 높이 | 빈틈 | 불탄 팬케이크 | 자동 회전 |
|---|---|---|---|
| 0–20 | 90° | 0% | 없음 |
| 20–50 | 70° | 15% | 없음 |
| 50–100 | 55° | 25% | 5°/s |
| 100+ | 45° | 35% | 10°/s |

```
│   └── scenes/
│       ├── BootScene.js    # 프리로드 + 파티클 텍스처 + 폰트 대기
│       ├── MenuScene.js    # 타이틀 / 최고 점수 / 시작

│       └── GameOverScene.js# 결과 / 다시하기

└── (루트의 원본 아트 .jpg/.png)
```

## 🎨 에셋 파이프라인 (중요)

원본 아트는 모두 **초록색 크로마키 배경** 위에 그려져 있습니다(투명 PNG 아님).
`scripts/process_assets.py`(스크래치패드에 보관) 로 Python + Pillow 를 이용해:
1. 초록색을 제거하여 투명 처리(스필 억제 + 가장자리 페더링),
2. 여백 자동 크롭, 리사이즈(스프라이트 최대 512px, 배경 높이 1280px),
3. ASCII 이름으로 `assets/` 에 저장

배경 4종은 크로마키가 아니므로 그대로 리사이즈만 합니다.

> 원본 아트를 교체하면 그 스크립트를 다시 실행해 `assets/` 를 재생성하세요.
